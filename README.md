# Employee Management System (MERN Stack)

A full-stack Employee Management System with role-based dashboards for managers and employees.

## Features
- Attendance management with analytics
- Task assignment and tracking
- Leave request & approval workflow
- Role-based authentication
- Dark / Light mode UI

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

## Setup
1. Clone the repository
2. Install dependencies in `client` and `server`
3. Create `.env` files using `.env.example`
4. Run the application
